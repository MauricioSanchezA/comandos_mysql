COMANDOS MYSQL

Xampp

1. Abrir Xampp.
2. Iniciar MySQL (Click en el botón start).
3. Una vez este corriendo MySQL, click en el botón Shell.
4. Digitamos el siguiente comando (usuario de MySQL):
mysql -u root
5. Dar enter.

Comandos Inicio

6.(muestra las bases de datos creadas): SHOW DATABASES;
7. (crear la base de datos. Ejemplo es el nombre de la bd): CREATE DATABASE bd1_2560119;
8. (Selecciona la bd con la que se va a trabajar): USE bd1_2560119;
9.(muestra las tablas de la bd): SHOW TABLES;

Inicio --->

Comandos Crea, Grafica, Ingresa, Selecciona

10. (crea la tabla):
CREATE TABLE Usuarios(id_usuarios AUTO_INCREMENT, primer_nombre VARCHAR (44) NOT NULL, PRIMARY KEY (id_usuarios));
11. (grafica la tabla): DESC Usuarios;
12. (ingresa datos a la tabla): INSERT INTO Usuarios(primer_nombre) VALUES ('Juan'),('Laura'),('Maria');
13. (selecciona todos los campos de la tabla)
SELECT * FROM Usuarios;

Inicio --->

Comandos Seleccionar, Ordenar Desc y Asc

14. (seleccionar un campo de la tabla):
SELECT primer_nombre FROM Usuarios;
15. (selecciona todos los campos de la tabla y los ordena de forma descendente):
SELECT * FROM Usuarios ORDER BY id_usuarios DESC;
16. (selecciona todos los campos de la tabla y los ordena de forma ascendente):
SELECT * FROM Usuarios ORDER BY primer_nombre ASC;

Inicio --->

Comandos Modifica, Seleccionar, Actualizar

17. (modifica la tabla para agregar una columna):
ALTER TABLE Usuarios ADD COLUMN primer_apellido VARCHAR (100) NOT NULL DEFAULT ('Apellido');
18. (verificamos que la columna este agregada en la tabla):
SELECT * FROM Usuarios;
19. (actualizamos un campo de la tabla):
UPDATE Usuarios SET primer_apellido='Perez' WHERE id_usuarios=2;

Inicio --->

Comandos Actualizar y modificar 2 campos de registro

20. (verificamos que el campo se actualice):
SELECT * FROM Usuarios;
21. (actualizamos un campo de la tabla):
UPDATE Usuarios SET primer_nombre='Juliana', primer_apellido='Morales' WHERE id_usuarios=3;
22. (verificamos que el campo se actualice):
SELECT * FROM Usuarios;

Inicio --->

Comandos Actualizar, Verificar y Eliminar

23. (actualizamos un campo de la tabla):
UPDATE Usuarios SET primer_apellido='Martin'
WHERE id_usuarios=1;
24. (verificamos que el campo se actualice):
SELECT * FROM Usuarios;
25. (elimina el campo):
DELETE FROM Usuarios WHERE id_usuarios=2;
26. (verificamos que el campo se encuentre eliminado):
SELECT * FROM Usuarios;

Inicio --->

Comandos Agregar Columna, Sumar, Contar y Promedio

27. (agrega una columna y por defecto el valor del campo es 10):
ALTER TABLE Usuarios ADD COLUMN edadINT DEFAULT (10);
28. (suma los datos de un campo):
SELECT SUM(edad) FROM Usuarios;
29. (cuanta cuantos datos hay):
SELECT COUNT(id_usuarios) FROM Usuarios;
30. (promedio):
SELECT AVG(edad) FROM Usuarios;

Comandos parar, ruta, guardar copia y Eliminar BD

31. Comando Ctrl+C para parar la consola.
32. Nos ubicamos en la ruta en la que queremos hacer la copia de seguridad de la bd.
ej: c:\User\yuly\Desktop 33. (genera la copia de seguridad):
mysqldump -u root bd1_2560119 > bd1_2560119.>sql 34. (Elimina la Base de datos):
DROP DATABASE bd1_2560119;
